# BrandUp AgencyOS - Final Master Prompt

Build a production-ready SaaS-style Agency CRM and Client Management Web Application called **BrandUp AgencyOS** for BrandUp Digital Marketing.

The app must be fully editable, mobile responsive, premium, fast, and built for managing clients, leads, payments, follow-ups, tasks, content delivery, documents, reports, and Excel import/export.

## Design Style

- Premium SaaS dashboard
- Dark navy blue, white, teal, and gold accent theme
- Inspired by Stripe, Notion, Linear, HubSpot, ClickUp, Vercel
- Glassmorphism cards
- Clean typography
- Responsive mobile-first UX
- Light/Dark mode

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- ShadCN UI
- Supabase
- PostgreSQL
- React Query
- Zustand
- Recharts
- Framer Motion
- ExcelJS
- React Hook Form
- Zod Validation

## Core Modules

1. Authentication and role-based access
2. Dashboard with KPIs and charts
3. Client CRM
4. Lead CRM
5. Follow-up management
6. Payment tracker
7. Service management
8. Task and work management
9. Content calendar
10. Document vault
11. Client notes and activity timeline
12. Reports
13. Excel import/export
14. Global search
15. Notifications
16. AI assistant
17. Settings

## Most Important Requirements

- Every record must be editable.
- Clients, leads, tasks, payments, services, follow-ups, notes, and documents must support add, edit, delete, search, filter, and export.
- Export data to Excel and CSV.
- Import bulk clients, leads, tasks, and payments from Excel.
- Dashboard must show pending payments, today's follow-ups, overdue tasks, revenue, active clients, and leads.
- App must be deployable on Vercel with Supabase backend.
