# BrandUp AgencyOS - Necessary Files

This package contains the core planning and starter files needed to build BrandUp AgencyOS, a client CRM and agency management web app for BrandUp Digital Marketing.

## Included Files

1. MASTER_PROMPT.md - Final AI/developer prompt for full web app generation.
2. DATABASE_SCHEMA.sql - Supabase/PostgreSQL database schema.
3. APP_STRUCTURE.md - Suggested Next.js project structure and pages.
4. FEATURE_REQUIREMENTS.md - Detailed module-wise feature checklist.
5. EXCEL_IMPORT_EXPORT_SPEC.md - Excel import/export field mapping.
6. BRANDUP_CLIENT_CRM_TEMPLATE.xlsx - Editable Excel CRM template with sample clients.
7. .env.example - Required environment variables.
8. package.json - Recommended frontend dependencies.

## Recommended Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- ShadCN UI
- Supabase
- PostgreSQL
- ExcelJS
- Recharts
- React Hook Form
- Zod

## Main Goal

Build a fully editable web app where BrandUp can manage:

- Clients
- Leads
- Follow-ups
- Payments
- Tasks
- Services
- Content calendar
- Documents
- Reports
- Excel export/import
