# Upload Instructions

## For GitHub + Vercel
1. Extract ZIP.
2. Upload folder contents to a new GitHub repository.
3. Connect repository to Vercel.
4. Add environment variables from `.env.example`.
5. Deploy.

## For Supabase
1. Create Supabase project.
2. Open SQL Editor.
3. Paste and run `supabase/schema.sql`.
4. Add your Supabase URL and anon key in Vercel environment variables.

## For AI Coding Tools
Paste `docs/MASTER_PROMPT.md` and upload this whole folder. Ask the tool to complete every module using the provided schema and UI starter files.
