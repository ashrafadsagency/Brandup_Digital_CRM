# BrandUp AgencyOS

A GitHub-ready Next.js starter bundle for an agency CRM and operating system for BrandUp Digital Marketing.

## Modules Included
- Dashboard
- Client CRM
- Lead Management
- Payment Tracker
- Follow-Up Manager
- Task Manager
- Reports
- Projects
- Content Calendar
- Documents
- Team
- Excel export API example
- Supabase schema
- Seed client data
- Editable Excel CRM template

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- ExcelJS
- Recharts
- Framer Motion

## Quick Start
```bash
npm install
cp .env.example .env.local
npm run dev
```

Open `http://localhost:3000`.

## Supabase Setup
1. Create a Supabase project.
2. Open SQL Editor.
3. Run `supabase/schema.sql`.
4. Add keys to `.env.local`.

## Excel Template
See `templates/BRANDUP_CLIENT_CRM_TEMPLATE.xlsx`.

## Deployment
This project is ready for GitHub, Vercel, and Hostinger Node.js hosting.
